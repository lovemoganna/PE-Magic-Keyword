# 关键词管理后台使用指南

## 功能概览

关键词管理后台提供了完整的关键词数据管理功能，包括：

1. **CRUD操作** - 创建、读取、更新、删除关键词
2. **导入导出** - 支持JSON和CSV格式的数据导入导出
3. **多种预览模式** - 表格、网格、列表三种视图模式
4. **数据分析看板** - 统计图表和数据分析

## 访问方式

### 方式一：通过前台入口
在前台页面右上角点击"管理后台"按钮

### 方式二：直接访问
访问路径：`/admin`

## 功能详解

### 1. 关键词管理

#### 创建关键词
1. 点击右上角"新建关键词"按钮
2. 填写表单信息：
   - **关键词名称**（必填）：关键词的中文名称
   - **分类**（必填）：选择所属分类
   - **深度**（必填）：选择深度级别
   - **描述**（必填）：详细描述
   - **认知影响**：说明该关键词的认知影响
   - **示例**：添加使用示例（可多个，用逗号分隔）
   - **相关词**：添加相关概念（可多个，用逗号分隔）
   - **跨领域应用**：添加应用领域（可多个，用逗号分隔）
3. 点击"保存"按钮

#### 编辑关键词
1. 在关键词列表中点击"编辑"按钮（表格视图）或卡片上的"编辑"按钮
2. 修改需要更新的字段
3. 点击"保存"按钮

#### 删除关键词
1. 点击"删除"按钮
2. 确认删除操作

#### 批量删除
1. 勾选需要删除的关键词（支持全选）
2. 点击"批量删除"按钮
3. 确认删除操作

### 2. 搜索和筛选

#### 搜索功能
- 在搜索框输入关键词名称或描述内容
- 支持模糊匹配

#### 筛选功能
- **分类筛选**：按分类筛选关键词
- **深度筛选**：按深度级别筛选关键词
- 可组合使用多个筛选条件

### 3. 视图模式

#### 表格视图
- 适合快速浏览和批量操作
- 显示关键信息：名称、分类、深度、描述
- 支持全选和批量操作

#### 网格视图
- 卡片式展示，视觉效果好
- 适合浏览和快速预览
- 每个卡片显示完整信息

#### 列表视图
- 紧凑的列表展示
- 适合快速浏览大量数据
- 信息密度高

### 4. 预览功能

点击"预览"按钮可以查看关键词的完整信息，包括：
- 关键词名称和标签
- 详细描述
- 认知影响
- 示例列表
- 相关词列表
- 跨领域应用列表

### 5. 导入导出

#### 导出数据

**JSON格式**
- 点击"导入/导出" → "导出"标签
- 点击"JSON格式"卡片
- 自动下载JSON文件
- 包含完整的数据结构，适合程序处理

**CSV格式**
- 点击"导入/导出" → "导出"标签
- 点击"CSV格式"卡片
- 自动下载CSV文件
- 适合在Excel等表格软件中查看和编辑

#### 导入数据

1. 点击"导入/导出" → "导入"标签
2. 选择导入选项：
   - **合并模式**：将导入数据与现有数据合并
   - **覆盖已存在**：如果关键词已存在，用导入数据覆盖
3. 选择文件或粘贴数据：
   - **选择文件**：点击文件选择按钮，选择JSON或CSV文件
   - **粘贴数据**：直接在文本框中粘贴JSON或CSV数据
4. 点击"导入"按钮
5. 查看导入结果：
   - 成功数量
   - 失败数量
   - 错误信息（如有）

**导入格式要求**

JSON格式：
```json
[
  {
    "term": "关键词名称",
    "category": "core-catalysts",
    "depth": "foundational",
    "description": "描述内容",
    "examples": ["示例1", "示例2"],
    "related": ["相关词1", "相关词2"],
    "cognitiveImpact": "认知影响",
    "crossDomains": ["领域1", "领域2"]
  }
]
```

CSV格式：
```csv
term,category,depth,description,examples,related,cognitiveImpact,crossDomains
关键词名称,core-catalysts,foundational,描述内容,"示例1; 示例2","相关词1; 相关词2",认知影响,"领域1; 领域2"
```

### 6. 数据分析看板

点击"数据分析"标签查看统计信息：

#### 总览统计
- **总关键词数**：当前系统中的关键词总数
- **平均示例数**：每个关键词平均包含的示例数量
- **平均相关词数**：每个关键词平均包含的相关词数量
- **平均跨领域数**：每个关键词平均包含的跨领域应用数量

#### 分类分布
- 显示各分类的关键词数量和占比
- 可视化进度条展示
- 按数量排序

#### 深度分布
- 显示各深度级别的关键词分布
- 卡片式展示，包含数量和占比

#### 数据质量指标
- **完整度**：包含完整信息的关键词占比
- **丰富度**：平均关联信息数量
- **覆盖度**：覆盖的分类数量

## 数据存储

所有数据存储在浏览器的 `localStorage` 中，键名为 `magic_keywords_data`。

**注意事项**：
- 数据仅存储在本地浏览器
- 清除浏览器数据会丢失所有更改
- 建议定期导出数据备份
- 不同浏览器/设备之间的数据不共享

## 快捷键

- **Enter**：在输入框中按Enter键可快速添加项目（示例、相关词、跨领域）

## 最佳实践

1. **定期备份**：定期导出数据，避免数据丢失
2. **批量操作**：使用批量删除功能提高效率
3. **数据验证**：导入前检查数据格式是否正确
4. **分类管理**：合理使用分类和深度标签，便于管理
5. **完整信息**：尽量填写完整的字段信息，提高数据质量

## 故障排除

### 导入失败
- 检查数据格式是否正确
- 确认JSON格式有效
- 检查CSV分隔符是否正确

### 数据丢失
- 检查浏览器是否清除了localStorage
- 尝试从备份文件恢复

### 性能问题
- 如果关键词数量很大（>1000），考虑分批导入
- 使用筛选功能减少显示数量

## 技术支持

如遇到问题，请检查：
1. 浏览器控制台是否有错误信息
2. 数据格式是否符合要求
3. 浏览器是否支持localStorage
